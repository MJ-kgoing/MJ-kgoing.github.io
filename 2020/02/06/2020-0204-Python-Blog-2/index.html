2020-0204-Python-Blog-2

Posted on 2020-02-06

I just had my third python class and we were still talking about the List. The reason I’m now writing this blog is because for the first time, I have been in the “flow” that I truly and completely sink in to the process of creating code, finding errors and trying to fix it. This was led by an after-class homework assigned to us for this class that we were asked to create a program which contains certain functions.

I used around 20 minutes to figure it out and submit the first version. However, it was imperfect according to the teacher and he gave me some advices for improvement. Unfortunately, I failed to create the right program after numerous tries. During this process, I have been adjusting different part of the program while checking the content in the textbook. When these didn’t work, I seeked out for help.

​ (class discussion)

Unexpectedly, a lot of my classmates were enfacing the same problem as I. Consequently we started a class discussion even when it was already an hour later the end of class. Our teacher also joined after a while and with his aid I addresses the problem quickly and I was genuinely happy when I saw the expected output. I then shared my result in the chat and tried to answer questions from others. I was so into the process of solving problem that I didn’t even realize the pass of time.

​ (the program we were trying to building)

Now, I can understand why there are tons of internet worker who are willing to spend their whole day staring at their screen just for solving the bug. I can now admit, that coding is addictive. It is interesting and I wish it would be my new interest to establish further.
