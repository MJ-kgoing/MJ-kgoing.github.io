Python project-Mini Gene Finder

Posted on 2021-01-10

What is this project about

This is a partner project I completed in the school Computer Science class. We could choose from multiple projects based on the description given, and the challenge we did is called “Mini Gene Finder”, here is a brief introduction to the project that I wrote for the documentary part:

“The protein synthesis process is known as the central dogma of biology, in which transcription(transcribing the antisense strand of a gene to the mRNA strand) and translation(translating the mRNA into the genetic code of protein) takes place in order to decode the “recipe” of protein which is stored in the DNA. In this study, we aim to create a program that can model these two steps while also verifying whether the amino acid is valid by testing it with some basic rules. The target audience of this program is potentially the biologists who seek an efficient and accurate way to decode the protein synthesis process. The tool that will be used in this program is Repl.it which is a platform for share and build codes.”

What was the process

The project is divided into 3 criterias: B, C, and D:

Criteria B:

  • Difining the problem
  • Identify design specifications,
  • Break down the big program
  • Concieve and evaluate a design plan (pseudocode/ flowchart)

Criteria C:

  • Create schedule for building codes and evaluate
  • Actual program writing
  • Highlight parts of the code that demonstrates a high level of technical skill

Criteria D:

  • Present solution through screencast
  • Evaluate the strengths and weaknesses of the solution and suggest possible improvement
  • Evaluation on self contribution and achievements.
